\begin{abstract}
    For the purpose of familiarizing the team with low level programming of the microcontroller used during the course, a simple assembly program was written in ARM Thumb 2 assembly for the ARM Cortex-M3 microcontroller on a EFM32GG DK3750 board.
    The program controlled a series of LEDs using gamepad buttons.
    The usage of power-saving features of the microcontroller were iterativly introduced to the program.
    The development board facilitated taking power measurments between each step.
    The resulting solution makes full use of the deep sleep feature of the microcontroller to enable maximum power efficiency with the given hardware.
\end{abstract}
